Bellow are the detailed information about this study as well as its jupyter-notebooks:

Incoming data

These are data fetched to produce the datasets used by this analisys:

  • Rodrigo Pomber's JSON Pomber: worldwide whole convid-19 pandemic data by contries
  • ebwinter's [API] (https://github.com/novelcovid/api): contries geo-data
  • UN Population dataset: total population of each country (not all) provided by United Nations
  • brasil-estados.json: a JSON file cotaining all geographic data from all brazilian states from IBGE's API. These data are used to draw the states over the map of Brazil.
  • brazilian_states.csv: descriptive data from all the braizlian states from IBGE's API
  • gov_brazil_corona19_data.csv: updated pandemic data in Brazil provied by brazilian Ministery of Health.

Dataset

  • world_countries_2019.csv: a consolidated dataset from ebinters and UN data, which is used to normalize some pandemic indicadors by countries total population
  • world_corona19_data.csv: the main dataset used by this study. It is made from Pomber's JSON and enriched with the following features which are derivate to cases, deaths and recoveries, although, to simplify I will use just case's :
    • day: number of the day since the first case of contamination
    • cases: cumulative number of cases since day 0
    • case_day: number of the cases recorded in a specific day
    • %var_case_day: daily variation of cases from the previous day (in percent)
    • cases_million: number of daily cases per one million of population of a coumtry. It is better comparable along the countries
    • avg7_cases: moving average of the daily cases among the last 7 days
    • avg7_cases_million: moving average of the daily cases per one million among the last 7 days
    • perc_death: percentage of deaths over the cases of the country
  • brazil_corona19_data.csv: a previous-like dataset, but with brazilian pandemic data over its states.

Notebooks

Here are my jupyter-notebooks used to build this study:

countries_data_aggregator.ipynb: the aggregator of all countries data.

data_engineering.ipynb: data and feature engineering for the main dataset.

analysis.ipynb: all the analises and charts about the world and brazilian pandemic.

brazil_analysis.ipynb: all the analises and charts about the brazilian pandemic.

prediction.ipynb: all prediction of world and brazilian pandemic using Facebook-Prophet.

brazilian_prediction.ipynb: all prediction of brazilian pandemic among its states using Facebook-Prophet.
